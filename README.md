# FastAPI + Vue 3 To-Do List

A fullstack To-Do List project built with **FastAPI** (Python) for the backend and **Vue 3 + TypeScript** for the frontend.  
---

## Tech Stack

- **Backend:** FastAPI, SQLAlchemy
- **Frontend:** Vue 3, TypeScript, Vite, Pinia
- **Database:** MySQL 
- **Authentication:** JWT

---

## Project Status

- [x] Frontend skeleton created (Vue 3 + TypeScript)
- [x] Backend skeleton + MySQL connection
- [x] CRUD API for tasks
- [x] Authentication (JWT)
- [x] Tests (pytest)

## Getting Started (Frontend)

1. Go to frontend folder:
   ```bash
   cd frontend
3. Install dependencies:
4. ```bash
   npm install
5. Start dev server:
6. ```bash
   npm run dev
7. Open in your browser:
   http://localhost:5173
